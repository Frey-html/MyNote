##### 成功响应示例

如果用户存在，后端会返回 200 OK状态码和用户数据的 JSON 表示：
```json
{   "userId": 1,   
    "userName": "testUser",   
    "avatar": "avatarUrl" }`
```

##### 失败响应示例
如果用户不存在，后端会返回 404 Not Found状态码。
```json
{   "timestamp": "2024-05-19T14:15:22.123+00:00",          "status": 404,   
    "error": "Not Found",   
    "message": "No message available",   
    "path": "/api/forum/users/1" }`
```
